Safely navigate on Internet (1h)

The theme of this online course is : " Learn how to surf safely on the Internet".


This course deals with aspects of security on the Internet such as: never to enter the access code to his bank on an online form, the symbol "padlock" in the address bar which indicates that the data will be sent securely and that before being sent over the Internet, the bank data is encrypted. There are also protection mechanisms complementary as "3D Secure" or double identification via SMS code for example.

The second part of this course is about passwords. These must be very long and different on each site ideally as well as contain special characters. Using a password manager (a kind of safe-deposit box) can be useful for this. In general it is more optimal to use the latter option with double authentication.

The third part of this course addresses various danger. For example the scam, which makes the user believe that he will earn a lot of money and become rich. We also learn that only 25% of the emails sent are "true" and not spam (the other %) and that we must also be wary of the mails ladle from friends we know because it is easy to pretend to be someone else. You must also check suspicious mail information with hoax website such as hoaxbuster.
