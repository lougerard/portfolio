OOP with Python (3h)

The theme of this online course is : " Discover object-oriented programming with Python".


I learned object-oriented Python programming with this course through an little example : the manipulation of data from a "populated world". For example, to answer the question : From what population density is less pleasant than the average ? The final program therefore plot a graph of this result. This course covers object-oriented concepts such as : attributes, methods, sequence diagram, class diagram, object, class, constructor ... We also see the manipulation of JSON data.
